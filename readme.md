## install

```
$ npm install --global @geekjuice/lxy
```

## usage

```
$  lxy --help

  learn x in y minutes

  usage: lxy [thing]

  options:
    -t, --things   list things to learn
    -h, --help     show help
    -v, --version  show version

```
